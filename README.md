### 1. The Purple Patch (Adjective Bloat)

- **The Exploit:** Users could bypass the Butcher (who only hunted adverbs) by stuffing sentences with decorative adjectives (e.g., _"The massive, ancient, brooding, dark tower"_).
- **The Patch:** **Suffix Scanning**. The Physics Engine now detects adjectival patterns (`-ous`, `-ful`, `-ive`).
    - **Penalty:** Adjectives now contribute to the **Garnish Ratio** (Weight: 0.5). If you decorate a noun too heavily, Drag increases.

### 2. Deep Storage (Object Permanence)

- **The Problem:** The "Goldfish Horizon." The system previously forgot any object mentioned more than 10 turns ago.
- **The Patch:** **The Hippocampus**. Heavy Matter (Guns, Keys, Maps) is now buried in a persistent dictionary. The Muscaria can recall items from the very start of the session.

### 3. Axiom Injection (Reality Definition)

- **The Problem:** The logic engine was blind. It accepted "Dry Oceans" and "Cold Fire" because it didn't know the definitions of words.
- **The Patch:** **Property Inheritance**. The system now loads ~100 essential definitions across 5 dimensions (`THERMAL`, `LUMENS`, `HYDRATION`, etc.).
    - **Effect:** Contradictions (e.g., "Dark Sun") now trigger a **LOGIC TEAR**.

### 4. The Sweat Protocol (Kinetic Exemption)

- **The Problem:** The "Hydration Monitor" unfairly penalized high-energy, rhythmic writing as "FLOODED" because it used too many connectors.
- **The Patch:** **Sweat vs. Water**. If a text is "FLOODED" but maintains a high **Kinetic Ratio** (>25%), the penalty is reduced from **+4.0** to **+1.0**.
    - **Philosophy:** "You are allowed to sweat if you are running."

### 5. The Narcissism Patch (Mirror Trap v2)

- **The Exploit:** The Mirror Trap ("It is not X...") ignored the user talking about themselves.
- **The Patch:** **Self-Reflection**. The regex now targets `(i|we)` negation loops. You cannot waste narrative drag defining who you are *not*.
