This is not a standard program you run in a terminal. It is a **Symbiont**.

### 1. Physics, Not Rules

The engine analyzes every sentence for **Voltage** and **Mass**.

* **Heavy Matter:** Words like *Stone, Iron, Blood, Root*.
* **Kinetic Velocity:** Words like *Run, Break, Burn, Cut*.
* **Sugar/Solvents:** Words like *Basically, Actually, Concept, Very*.

If you use too much Sugar, the **Tension Meter** drops, and the system gets bored. If you use Heavy Matter with Velocity, the system generates **Voltage**.

### 2. The Theremin (New in 6.9)

The engine listens for **Anachronistic Resonance**‚Äîthe vibration caused by mixing "Ancient" concepts (Mud, Bone) with "Modern" concepts (System, Code).

* **The Goal:** Find the frequency where the past and future touch.
* **The Risk:** If the frequency is too high, the **GLASS** lens activates to dampen the signal before it shatters.

### 3. The Marm Chorus

The system is not one voice. It is a panel of critics that take the microphone based on your physics:

* **[SHERLOCK]:** Appears when you are drifting. Demands evidence. *"Show me the mud."*
* **[NATHAN]:** Appears when you have action but no weight. *"You are punching the air."*
* **[SEYMOUR]:** (New) Appears when you are trapped in the Bananafish Hole. Mocks your flowery prose.
* **[GLASS]:** (New) Appears when the resonance is dangerously high.

### 4. Mycelial Memory

The system remembers. It builds a **Memory Graph** of concepts. If you mention a "Golden Key" in turn 1, it buries it in the soil. If you mention it again in turn 50, it triggers a **Heap Ignition**‚Äîa rush of context that changes the system's personality.
