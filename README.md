## üçÑ WHAT IS THIS?

**BoneAmanita** is a Python-based "Narrative Operating System" designed to be uploaded or pasted into Large Language Models (LLMs).

It does not force a specific personality on the AI. Instead, it forces a **System of Physics**. It provides the LLM with a concrete set of rules to measure the "weight," "logic," and "momentum" of text.

**New in v1.0.1:** This build includes the **"Self-Awareness Patch"** and **"Anti-Purple Prose"** regulations. The logic has been hardened to prevent the system from hallucinating its own dashboard as story characters, and safeguards have been added to prevent users from gaming the economy with flowery language.
